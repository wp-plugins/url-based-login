== Description ==

URL Based Login allows you to directly login from an allowed URL. So if you want to allow someone to login but you do not want to share the login details just create a Login URL with URL Based Login and provide them the Login URL.

Features in URL Based Login 1.3.0  include:

* Create multiple Login URLs
* Choose the username accessible when accessed by the allowed URL
* Delete Login URLs
* Safe & Secure
* No passwords saved
* Completely FREE
* Licensed under GNU GPL version 3

== Installation ==


INSTALLATION

Easy Way:

Go to the WordPress Dashboard "Add New Plugin" section.
Search For "URL Based Login".
Install, then Activate it.

Old Way:

Download the plugin Unzip (if it is zipped) and upload url-based-login folder to the /wp-content/plugins/ directory
Activate the plugin through the 'Plugins' menu in WordPress

CONFIGURATION

The plugin's admin page can be found under Settings menu (Settings => URL Based Login)
